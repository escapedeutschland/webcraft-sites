Wann was: Verlauf = Standard (Instagram, E-Mail-Profil, überall). Navy = auf dunklem Grund. Einfarbig = wenn schlicht/ein Ton gewünscht (z. B. Stempel, Fax, einfarbiger Druck). Auf Weiß / hell = für helle Flächen ohne Kachel (Dokumente, Website-Inhalt). Die „Auf Weiß"-Vorschau steht hier auf Dunkel, damit man die weiße Grafik sieht.
